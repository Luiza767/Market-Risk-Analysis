# Market-Risk-Analysis
Market risk analysis with VaR modeling, backtesting validation, and stress testing under adverse scenarios.
# Market Risk Analysis Project

## Overview
This project demonstrates core market risk concepts:
- Value at Risk (VaR)
- Expected Shortfall (ES)
- Monte Carlo simulation
- Backtesting

## Structure
- var_model.py: calculates VaR & ES
- backtest.py: checks VaR accuracy

## Usage
Run:
python var_model.py
python backtest.py
